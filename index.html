Apa itu Progressive Web App (PWA)?

Progressive Web App (PWA) adalah jenis aplikasi berbasis web yang dirancang agar bisa ditampilkan dan berfungsi seperti aplikasi native.

Komponen Utama PWA

  • HTTPS ‒ koneksi aman.
  • Web app manifest ‒ definisi ikon & nama.
  • Service worker ‒ caching + offline.
